Project records audio from default mike source and stores it into file.

Run this project by below Command

/* Build project */ // replace with your visual studio version
cmake -H. -Bbuild	"Visual Studio 14 2015 Win64"

/* Compile project */
cmake --build build

// compilation adds .exe into Debug folder
/* Run Project */


I have created bat file to run program, just click on .bat file file will compile and execute automatically.
